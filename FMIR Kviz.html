FMIR Kviz - Napredne opcije

Učitajte svoja pitanja iz JSON fajla sa računara

Uputstvo za korišćenje

  1. Pripremite JSON fajl sa pitanjima u ispravnom formatu (pogledajte primer ispod)
  2. Kliknite na dugme "Odaberite JSON fajl" i odaberite svoj fajl
  3. Kada se fajl učita, kliknite na "Pokreni kviz"
  4. Odgovarajte na pitanja i pratite svoj napredak

Format JSON fajla:

[
  {
    "question": "Menične radnje nisu:",
    "options": [
      { "text": "akceptiranje", "correct": false },
      { "text": "certifikacija", "correct": true },
      { "text": "opoziv", "correct": true },
      { "text": "amortizacija", "correct": false },
      { "text": "jemstvo (avaliranje)", "correct": false }
    ],
    "explanation": "Menične radnje uključuju akceptiranje, jemstvo (avaliranje), ali ne i certifikaciju ili opoziv."
  },
  {
    "question": "Koji od finansijskih izveštaja nisu ključni:",
    "options": [
      { "text": "glavna knjiga", "correct": true },
      { "text": "bilans uspeha", "correct": false },
      { "text": "dnevnik", "correct": true },
      { "text": "izveštaj o novčanim tokovima", "correct": false },
      { "text": "izveštaj o prometu sredstava", "correct": true }
    ],
    "explanation": "Ključni finansijski izveštaji su bilans stanja, bilans uspeha i izveštaj o novčanim tokovima."
  }
]
Odaberite JSON fajl sa pitanjima sa vašeg računara
